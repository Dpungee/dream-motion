# 🎬 Dream Motion

AI-powered text-to-video generator using fal.ai API. Transform text prompts into lifelike AI-generated videos.

## Features

- **4 AI Video Models**: Wan 2.1, Hunyuan Video, Stable Video, Kling 2.1
- **AI Prompt Enhancement**: Claude-powered prompt improvement
- **6 Visual Styles**: Cinematic, Anime, Photorealistic, and more
- **Aspect Ratio Controls**: 16:9, 9:16, 1:1, 4:3
- **Real-time Queue**: Progress tracking with generation history
- **Dark UI**: Purple gradient theme with particle effects

## Usage

1. Get a free API key from [fal.ai](https://fal.ai/dashboard/keys)
2. Open the app and enter your API key
3. Type a prompt and generate videos
